# url62

url62 is a library to generate really unique and url friendly IDs
based on UUID and base62

## Usage

```
user=> (url62.core/generate-id)
"U7PVVMkSSJGepn8RKilghA"
```

leiningen:

```
[url62 "1.0.0-SNAPSHOT"]
```
